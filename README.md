# Ray cast arduino
tested on a TFT display with a 1.8-inch ST7789 driver

Connection:
  Display:
    GND        -> GND
    CS         -> 10
    RESET      -> 8
    A0 (DC)    -> 9
    SDA (MOSI) -> MOSI
    SCK        -> SCK
    LED (VCC)  -> 3.3V
  Joystick:
    VCC -> 5V
    GND -> GND
    SW -> A3 | A4
